üîÑ Data Flow

1. User Action
User clicks Login button in LoginScreen
2. State Management
AuthCubit.login() is called, emits loading state
3. Use Case
LoginUseCase validates input and calls repository
4. Repository
AuthRepositoryImpl calls remote data source
5. Data Source
AuthRemoteDataSource makes HTTP request via ApiClient
6. API Response
Backend returns JSON, converted to LoginModel
7. Mapping
LoginMapper converts Model ‚Üí Entity
8. Return Flow
Either<ErrorMsg, LoginEntity> flows back through layers
9. UI Update
AuthCubit emits success, UI updates and navigates
